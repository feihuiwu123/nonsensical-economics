# 荒谬经济学 (Nonsensical Economics)

一个分析金融市场中反直觉现象的文档集合，重点关注加密货币和股票市场。

## 概述

本仓库记录了金融市场中持续存在的“荒谬”或反直觉结构——那些看似不合理但由于底层激励、信息不对称和行为偏差而继续存在的模式。分析遵循一致的框架，识别：

- **反直觉现象**
- **谁受益**
- **谁受损**
- **获利工具/手段**
- **为什么这种结构尽管明显不合理却持续存在**

## 内容示例

### 加密货币市场 (28个案例)

加密货币分析涵盖三个层面的28个反直觉结构：

1. **比特币层 (8个案例)**
   - 没有现金流，却有万亿美元市值
   - 波动极大，却被叫“储值工具”
   - 不适合高频支付，却一直被叫“货币”
   - 减半是公开信息，却还能年年被当利好炒
   - 烧大量电，不产出实体商品，却被视为“有价值”
   - 链上越堵、手续费越贵，市场反而当成“繁荣”
   - 有人忘了私钥，反而对剩余持币者是利好
   - 讲去中心化，但持币、算力、托管其实都很集中

2. **基础设施层 (8个案例)**
   - 去中心化资产，主要靠中心化交易所定价
   - 稳定币不是银行存款，却成了整个币圈美元底座
   - “稳定币”最怕的就是不稳定
   - 链上透明不等于公平，反而更容易被抢跑
   - 自托管理论上最安全，实务上很多人反而更危险
   - 现货只是门面，真正的大抽水在合约和永续
   - 方向看对了，也可能照样亏死
   - ETF 带来合法性，同时带来更强的中心化托管

3. **山寨币/DeFi/平台币层 (12个案例)**
   - 治理代币有投票权，却没有真正的利润权
   - 产品很火，不代表代币会涨
   - 高 APY 往往不是高收益，而是高补贴
   - 空投本想公平分配，最后工作室和女巫拿走最多
   - 低流通高 FDV：表面稀缺，实则未来瀑布
   - 上大所本该是开始，实际上常是退出窗口
   - 最没基本面的 meme，反而常常涨得最好
   - “代码即法律”只对小户成立，出大事照样人治
   - 跨链是为提升效率，结果成了最大失血口之一
   - 质押收益看起来像“利息”，其实很多是通胀转移
   - KOL 教你暴富，自己却主要靠广告、返佣和喊单赚钱
   - 顶级 VC、审计、明星背书，照样能炸

### 股票市场 (40+ 个案例)

股票市场分析识别了五个类别中的40多个反直觉现象：

1. **公司、融资与股东关系类**
2. **制度、规则与市场结构类**
3. **信息、中介与专业话术类**
4. **估值、题材与资金偏好类**
5. **纯心理层面的“没人逼你，但总会发生”类**

每个案例都审视了结构性激励以及声明目的与实际结果之间的持续错配。
